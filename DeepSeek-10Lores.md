## Comment Utiliser ces Lores

**1. Enquête Simple :**
```bash
searchlores investigate "L'intelligence artificielle révolutionnera l'éducation." --lore lores/education_critique.lore
```

**2. En Python pour une Analyse Multi-Lores :**
```python
from searchlores.core.engine import InvestigationEngine
from searchlores.lore.loader import load_lore

engine = InvestigationEngine()
lores = [
    "lores/education_critique.lore",
    "lores/techno_solutionisme.lore",
    "lores/epistemologies_alternatives.lore"
]

prompt = "L'IA personnalisée va résoudre les problèmes d'apprentissage."

for lore_path in lores:
    lore = load_lore(lore_path)
    context = engine.run(prompt, lore=lore)
    print(f"\n--- Analyse avec {lore.metadata.name} ---")
    print(context.findings.get("assumptions", []))
    print(context.findings.get("questions", []))
```

**3. Créer un Lore-Pack pour une Enquête Spécifique :**
Combinez des lores pour une investigation multi-couches. Par exemple, pour analyser un discours sur la "transition énergétique", utilisez `climat_et_pouvoir.lore` + `techno_solutionisme.lore` + `consensus_scientifique.lore`.

## L'Esprit Fravien dans Chaque Lore

Chaque fichier incarne la **méthode Fravienne** :

*   **Scepticisme radical** : Aucune évidence n'est acceptée.
*   **Remontée aux sources** : Pas de "quoi" sans "d'où" et "pour qui".
*   **Lateral thinking** : Les questions déplacent le regard.
*   **Anti-autoritarisme** : L'expertise est un objet d'enquête, pas une garantie.
*   **Engagement épistémologique** : La recherche est un acte politique, éthique et cognitif.

> *"The search for truth is not about finding the answer, it's about dissolving the question."* — Fravia (paraphrase)

Ces Lores sont des **générateurs de questions**, pas des bases de réponses. Ils sont conçus pour vous aider à creuser des puits de réflexion, pas pour vous donner à boire une eau pré-digérée.
